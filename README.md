# Sales-Analysis
Excel-based dashboard that transforms raw sales data into actionable insights through dynamic visualizations and filtering tools.

📌 Overview

This project presents an interactive Sales Dashboard built in Microsoft Excel to analyze and visualize sales performance.
The dashboard provides clear insights using charts, slicers, and dynamic reports to help in decision-making.

🎯 Objectives
Track overall sales performance
Analyze sales by region, category, and time
Create an interactive dashboard using slicers
Present data in a clear and visual way
📂 Dataset

The dataset includes:

Order ID
Product / Category
Sales Amount
Quantity
Region / City
Date (Year, Quarter, Month)
🛠️ Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Conditional Formatting
📊 Dashboard Features
📌 Total Sales Overview
🌍 Sales by Region
📦 Sales by Category
📅 Sales Trend Over Time
🎛️ Interactive Slicers (Year / Quarter / Category)

⚙️ How It Works
Data is organized in a clean table format
Pivot Tables are created for analysis
Pivot Charts visualize the data
Slicers are added for interactivity
Dashboard layout is designed for clarity
🚀 How to Use
Download the Excel file
Open it using Microsoft Excel
Use slicers to filter data
Explore different insights dynamically
📌 Key Insights
Identifies top-performing regions and products
Highlights sales trends over time
Helps in comparing categories easily
Provides quick decision-making support
